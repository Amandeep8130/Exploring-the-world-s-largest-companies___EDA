
# EDA: World's Largest Companies Dataset

## Project Overview
This repository contains an **Exploratory Data Analysis (EDA)** of the *World’s Largest Companies* dataset. The purpose of this project is to analyze and visualize the financial and structural characteristics of the largest corporations worldwide.  

The analysis focuses on:
- Identifying trends among top global companies  
- Comparing industries and regions  
- Exploring relationships between financial metrics  
- Highlighting outliers and unique patterns  

---

## Dataset
- **Source**: Public dataset of the world’s largest companies (e.g., Forbes Global 2000, Fortune Global 500, or equivalent)  
- **Features include**:
  - Company Name  
  - Country / Region  
  - Industry / Sector  
  - Market Capitalization  
  - Revenue  
  - Profits  
  - Assets  
  - Number of Employees  

---

## Methodology
The EDA process includes:
- **Data Cleaning**: Handling missing values, duplicates, and formatting issues  
- **Descriptive Statistics**: Summary measures of financial metrics  
- **Visualization**:
  - Distribution plots (histograms, boxplots)  
  - Industry and country breakdowns (bar charts, treemaps)  

- **Insights & Findings**: Key takeaways about industries, regions, and company performance  

---

## Key Insights
- Top industries dominating the global corporate landscape  
- Regional distribution of the largest companies  
- Market capitalization vs revenue relationships  
- Outliers with unusual financial structures
